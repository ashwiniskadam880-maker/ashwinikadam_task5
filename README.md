📚 Digital Library Management System

A simple Java console-based application to manage books in a digital library.

 🚀 **Features**

* Add new books
* View all books
* Search books by title
* Easy menu-driven interface

## 🛠️ Technologies Used

* Java
* ArrayList
* Scanner
* Object-Oriented Programming (OOP)

## 📋 How to Run

1. Clone the repository:

   ```bash
   git clone <https://github.com/ashwiniskadam880-maker/ashwinikadam_task5.git>
   ```

2. Open the project in your Java IDE or terminal.

3. Compile the program:

   ```bash
   javac LibraryManagement.java
   ```

4. Run the program:

   ```bash
   java LibraryManagement
   ```

## 📖 Menu Options

```text
1. Add Books
2. View Books
3. Search Book
4. Exit
```
